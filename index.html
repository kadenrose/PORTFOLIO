Case Studies

Objective

Design and develop a campaign website that balances creativity with industry-standard usability—capturing attention through a compelling visual identity while clearly conveying key information to the public.

Process

I began by planning the site’s information architecture through research on established campaign websites. By examining the content and structure of sites belonging to well-known political figures, I determined that the website should include four key pages: Home, About, Events, and Get Involved. Each page was designed with clear user goals in mind—such as providing accessible forms for engagement and participation.

After creating wireframes and finalizing the layout, I built the site using HTML, CSS, and JavaScript, focusing on usability and accessibility. I implemented interactive features like a real-time election countdown, a light/dark mode toggle, and volunteer form validation. Throughout the process, I applied responsive design principles to ensure consistent performance across all devices.

Version control was managed through GitHub. The process emphasized clean, semantic code, user-centered design, and practical problem-solving—reflecting how I approach all web development projects from concept to completion.

Results

The front-end included working forms, responsive layouts, and dynamic JavaScript features. The design effectively communicated the campaign’s message and guided users toward key actions such as donating and volunteering.

Objective

The client needed to launch a new product line paired with a brand “story” that reflected the company’s core values and narrative.

Process

Within the team, I was responsible for handling the site’s functionality. I developed all JavaScript features and user interactions, including form validation, collapsible menus, and shopping cart logic.

The project addressed a common need among outdoor retail brands: creating an engaging, functional e-commerce platform that not only sells products but also communicates the personality and values of the brand’s community.
The three-week development period began with the creation of a team charter that outlined responsibilities and communication expectations. During the kickoff stage, we shared concepts and development strategies, then began hard-coding the site’s structure and layout. Once the UX team presented their finalized design, we moved into our individual responsibilities, and I began implementing the site’s functionality using JavaScript.
Throughout each revision round, I addressed feedback through debugging and also assisted other teammates outside my direct role. By the final presentation, our team delivered a working eCommerce prototype that demonstrated both technical precision and strong collaborative problem-solving.

Results

Prototype successfully showcased the site’s functionality, structure, and visual direction.

The goal of this project was to create a biography website highlighting the career and impact of professional skateboarder Rodney Mullen. The site was built with a mobile-first approach to ensure seamless usability across devices and multiple interactive elements into a cohesive user experience.

Process

I began by structuring the site with HTML and Bootstrap to ensure a clean, responsive foundation. After establishing the layout, I researched Rodney Mullen’s career, collecting accurate biographical details and high-quality images to bring authenticity to the content.
Once the structure and content were in place, I moved into development, implementing jQuery and vanilla JavaScript to add interactivity. This included building accordion sections for biographical milestones, a carousel for visual storytelling, and API integration for dynamic data. Each feature was tested extensively throughout development to ensure smooth performance and a truly mobile-first experience.

Results

The completed biography website successfully showcased Rodney Mullen’s life and legacy through an engaging, mobile-first design. The site delivered smooth, responsive interactivity using jQuery and vanilla JavaScript, including fully functional accordions, image carousels, and memory storage features.
